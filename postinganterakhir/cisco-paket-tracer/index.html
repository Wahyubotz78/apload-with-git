Koneksikan jaringan dengan 3 Router dan 3 Pc menggunakan Cisco Paket Tracker !

Jawab :

  • Merancang tampilan


  • Melakukan konfigurasi alamat ip address dan gateway untuk komputer pertama


  • Melakukan konfigurasi alamat ip address dan gateway untuk komputer kedua


  • Melakukan konfigurasi alamat ip address dan gateway untuk komputer ketiga


  • Melakukan konfigurasi ip address pada FastEthernet0/0 untuk router pertama


  • Melakukan konfigurasi ip addess pada serial 2/0 untuk router pertama


  • Melakukan konfigurasi ip addess pada FastEthernet0/0 untuk router kedua


  • Melakukan konfigurasi ip addess pada serial 2/0 untuk router kedua


  • Melakukan konfigurasi ip addess pada serial 3/0 untuk router kedua


  • Melakukan konfigurasi ip addess pada serial 2/0 untuk router ketiga


  • Melakukan konfigurasi ip addess pada serial 3/0 untuk router ketiga


  • Melakukan RIP Routing untuk router pertama


  • Melakukan RIP Routing untuk router kedua


  • Melakukan RIP Routing untuk router ketiga


  • Nampak pada gambar setiap komputer saling bertukar pesan dan semuanya bersatatus Successfull
