Macam-Macam Struktur Dasar Html

Pada html terdapat struktur yang dinamakan struktur dasar html yang terdiri dari tag doctype, html, head, dan body. Namun sebenarnya struktur dasar pada html bukan hanya itu, tetapi ada juga meta, tittle, dan link.

Penjelasan Struktur Dasar Html


  1. Deklarasi Html5

    2. Doctype atau Document Type Decaration (DTD) adalah suatu instruksi kepada web browser bahwa code kita ditulis dengan html5.



  2. Elemen
    3.

    Elemen adalah suatu komponen-komponen dokumen html yang terdiri dari tag-tag atau atribut. Seperti pada gambar di bawah, garis merah yang melingkar, di dalamnya adalah elemen struktur dasar pada html.



  3. Tag Pembuka dan Tag Penutup

    4. Tag pembuka di tandai dengan <> dan tag penutup di tandai dengan < / >, contoh tag pembuka dan penutup pada struktur dasar html.

    • Yang di garis berwarna hijau dijelaskan bahwa Tag html adalah tag yang berfungsi sebagai pembungkus semua codingan kita mulai dari head sampai ke body
    • Yang di garis berwarna merah dijelaskan bahwa Elemen head adalah elemen yang berfungsi untuk membungkus meta tag, title, dan link pada struktur dasar html
    • Yang di garis berwarna hijau dijelaskan bahwa Title adalah tag elmen html yang berfungsi untuk menulis judul pada web kita yang nantinya akan tampil di mesin pencarian google.

      • Judul yang kita tulis di tag title maka akan di tampilakan di browser seperti gambar di bawah.


    • Yang digaris berwarna biru dijelaskan bahwa Tag body berfungsi sebagai wadah untuk codingan-codingan kita yang nantinya code tersebut adalah perintah apa yang ingin kita tampilkan di halaman website kita nanti.

      • Sebenarnya masih banyak tag-tag yang memiliki pembuka dan penutup, akan tetapi di sini kita hanya mempelajari struktur dasar html jadi saya hanya menjelaskan tag yang ada pada codingan html di atas.


  4. Tag Yang Tidak Memiliki Penutup

    5. Codingan yang di lingkari warna putih adalah tag yang tidak memiliki penutup atau disebut dengan Self Closing Tag.

    Sama seperti tag yang memiliki pembuka dan penutup, tag self closing tag juga ada banya sebenarnya. Sebagian kecil contohnya adalah tag img, tag ini berfungsi untuk menambahkan gambar ke dalam halaman website kita.


Kesimpulan

  • Struktur dasar html yang terdiri dari tag doctype, html, head, dan body, meta, tittle, dan link.
  • Doctype atau Document Type Decaration (DTD) adalah suatu instruksi kepada web browser bahwa code kita ditulis dengan html5.
  • Judul yang kita tulis di tag title maka akan di tampilakan di browser seperti gambar di bawah.
